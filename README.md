## Setup and Usage

This is a static website, so no complex setup or installation is required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Rachit0910d/CODSOFT.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd CODSOFT
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in your web browser to view the portfolio.
